# موقع فيسبوك كلون - Facebook Clone

## كيفية التشغيل

1. **افتح الملف**: انقر مرتين على `facebook-clone.html` أو افتحه في متصفحك المفضل
2. **استمتع بالموقع**: الموقع يعمل بالكامل بدون الحاجة لخوادم أو إعدادات إضافية

## المميزات الرئيسية

### 🎨 التصميم
- تصميم عصري وجذاب مع تدرجات لونية جميلة
- واجهة متجاوبة تعمل على الهواتف والحواسيب
- دعم كامل للغة العربية مع اتجاه النص من اليمين لليسار

### 🚀 المميزات
- **الرأس**: شعار فيسبوك، مربع بحث، أيقونات التنقل
- **الشريط الجانبي الأيسر**: قائمة بالصفحات والمميزات
- **خلاصة الأخبار**: منطقة إنشاء المنشورات والقصص والمنشورات
- **الشريط الجانبي الأيمن**: قائمة الأصدقاء والمجموعات المقترحة
- **المراسلة**: نافذة محادثة تفاعلية

### ⚡ التفاعلية
- إعجاب وتعليق ومشاركة المنشورات
- إنشاء قصص ومنشورات جديدة
- محادثات في الوقت الفعلي
- إشعارات حية
- تأثيرات بصرية سلسة
